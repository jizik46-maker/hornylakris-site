Privacy Policy

This Privacy Policy applies to the application AutoPosterLakris operated by HornyLakris. The application integrates with the Pinterest API to allow managing and publishing Pins on behalf of the user.

Data Collection and Use

We do not collect, store, or share personal data outside of what is provided via the official Pinterest OAuth 2.0 flow. Access is limited strictly to the scopes granted by the user (e.g., boards:read, boards:write, pins:read, pins:write). No personal data is sold or transferred to third parties.

Authentication and Security

Authentication relies on Pinterest OAuth 2.0. Access tokens are used solely to enable features such as creating and managing Pins and are stored securely for this purpose.

Third-Party Services

This application communicates only with the official Pinterest API and does not integrate with other external services.
